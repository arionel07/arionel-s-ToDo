# ToDo App (Practice Project)

Простой ToDo-лист на **vanilla JavaScript, HTML и CSS**.

## Что реализовано сейчас
- Добавление задач
- Удаление задач
- Отметка задач как completed
- Фильтрация (all / active / completed)
- Сохранение задач в `localStorage`

## Зачем этот проект
Учебный проект для практики:
- работы с DOM
- событий (`click`, `keydown`)
- состояния приложения
- `localStorage`
- базовой архитектуры данных
